## Dashboard Reference Codebases

The `references` folder contains reference codebases for the E2B dashboard and
Trigger.dev dashboard. Use them as reference points for our dashboard
implementation.

When working on dashboard implementation, always consult these references for:
- UX and visual design patterns
- Core dashboard logic
- State management patterns
- Routing, data loading, and mutation flows
- Component structure and composition
- Error, loading, and empty states
- Any other useful implementation parameters present in the reference codebases

Prefer patterns proven in these reference projects when they fit our dashboard's
requirements.
